# 视图

虚拟表,将查询结果 保存为一个虚拟表

  • 创建视图 create view 视图名称 [(列名列表)] as 查询语句;
  • 查询视图 select * from 视图名称;
  • 修改视图数据 update 视图名称 set 列名=值 where 条件; 注意修改虚拟表的数据 原表数据也会修改
  • 修改视图 alter view 视图名称 (列名列表) as 查询语句;
  • 删除视图 drop view [if exists] 视图名称;
